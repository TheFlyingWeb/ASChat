Начиная с версии 1.3 Вы можете блокировать доступ для различных IP адресов, а также подсетей.

Для этого Вам необходимо перейти на страницу "Блокировка IP адресов" в Контрольной панели и ввести в поле адреса, которым Вы желаете ограничить доступ к сервису.

Пример IP адресов:

127.0.0.1 - это полный адрес
192.168.2 - это подсеть
